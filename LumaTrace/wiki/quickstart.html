WORKFLOW

从安装到第一次截图

保留原教程内容，但改成更清晰的步骤卡片和真实界面截图。

1

安装后先看菜单栏

启动 LumaTrace 后确认菜单栏图标存在，这是截图、录屏、设置和权限入口。

2

完成首次权限

按权限检查窗口引导开启屏幕录制和辅助功能，避免黑屏或长截图不能滚动。

3

第一次截图

用快捷键唤起截图界面，框选区域后直接复制、保存、标注或 OCR。

第一次截图

框选完成后底部工具栏会出现，常用操作集中在同一个位置。

  • Esc：取消截图。
  • Enter / ⌘C：复制当前截图。
  • ⌘S：保存到文件。

第一次设置建议

建议先检查外观、权限状态、默认格式和保存目录。

  • 默认图片格式建议 PNG。
  • 录屏格式根据使用场景选择。
  • 如果要 OCR / 翻译，再配置 Provider 和 API Key。
