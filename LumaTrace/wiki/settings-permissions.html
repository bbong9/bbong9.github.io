设置与权限 全部模块化

常规、快捷键、保存格式、授权、更新和高级能力都按模块化方式展示，图片对应到真实界面。

设置页分区

设置页按使用频率分区，常规和权限放在最前面。

  • 常规：启动、外观、权限状态和常用入口。
  • 快捷键：全局快捷键、截图界面功能键。
  • 保存与格式：图片格式、录屏格式和保存目录。

快捷键

所有快捷键集中查看，避免工具切换后仍停留在上一个工具。

  • 全局快捷键用于唤起截图。
  • 截图界面功能键只在截图框出现后生效。
  • 冲突时换成 ⌥A、⌘⌥2 等组合。

保存与格式建议

图片、录屏、自动保存和目录都在同一个页面调整。

  • 图片建议 PNG，保证标注清晰。
  • 录屏按场景选择格式。
  • 默认目录建议固定，方便历史记录查找。

高级设置说明

OCR、翻译、长截图、诊断相关能力集中在高级页。

  • API Key 和接口按 Provider 填写。
  • 长截图参数用于特殊网页兼容。
  • 诊断信息用于定位异常。
