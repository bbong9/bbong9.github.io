v2.0 - 2026-06-24

重点：区域录屏实时标注进入最终视频，设置页完成一次信息架构和排版清理，自动更新链路同步到 2.0。

录屏实时标注闭环

  • 区域录屏过程中绘制的矩形、画笔、箭头会写入最终视频。
  • 录屏控制条、标注工具栏和选区辅助层继续排除在录制内容之外。
  • 保留录屏演示所需的点击、高亮和按键提示能力，减少后期重新编辑。

设置页重构

  • 授权合并到“关于”顶部，自动升级排到其下方，关于信息放到最底部。
  • 删除通用页重复自动升级入口和未上线的云端微信同步入口。
  • 快捷操作配置全部展开，修复预览重叠和“恢复默认”被遮挡的问题。
  • 权限页按钮改为一列排列，首页启动按钮缩小并和周围内容对齐。
  • 统一导入配置、立即清理、权限操作等按钮高度、宽度和对齐方式。
  • 设置界面删除开发参考来源文案，只保留用户能理解的功能作用。

截图、OCR 与快捷操作

  • 补充窗口元素识别、系统截图快捷键检测和 OCR 文本规整支持。
  • 增加敏感信息遮罩能力，为后续隐私处理保留统一入口。
  • 优化 Quick Access 拖拽、触控板横扫、卡片状态保持和贴图联动。

发布包

  • 发布正式版 LumaTrace-2.0.dmg，构建号 74。
  • 同步 LumaTrace-latest.dmg、Sparkle appcast.xml 和 LumaTrace-2.0.zip。
  • 自动更新服务器同步 2.0，旧版用户可通过检查更新升级。

v1.12 - 2026-06-23

重点：长截图改为更接近微信的录屏式流式采集，只追加向下新增内容，回滚、停顿、抖动和不可靠帧不会污染最终图片。

长截图流式采集重构

  • 手动长截图启动后持续采集选区画面，用户直接滚动页面，右侧预览实时增长。
  • 只以最后一次成功追加的底部帧作为拼接锚点，来回滚动不会重置锚点。
  • 反向滚动会被识别为回滚帧，只跳过不入库，继续向下超过历史底部后再追加。

折叠与拼接稳定性

  • 增加原图二次位移校验，缩略分析和原图判断差异过大时跳过当前帧。
  • 小于稳定阈值的 10px、20px 级碎片切片不再写入最终图，减少动效、懒加载和视差页面上的羽毛状折叠。
  • 继续清理静态边缘、浏览器状态气泡和拼接缝，降低横线、黑边和重复浮层残留。

性能与体验

  • 使用 ScreenCaptureKit 流式帧源，减少频繁隐藏/显示窗口导致的闪烁。
  • 限制分析帧和预览图尺寸，降低 CPU 峰值和内存占用。
  • 长截图分段上限提升到 24 段，适合更长页面。
  • 工具栏、预览窗和蓝色边框继续不会进入导出结果。

验证

  • 新增回滚场景测试，覆盖向下滚动、回滚、回到旧位置、继续向下超过历史底部的流程。
  • 自动化测试通过 164 项。

v1.11 - 2026-06-22

重点：截图工具栏和贴图标注的粗细/字号改为 Command + 鼠标滚轮无级调节，并降低滚轮灵敏度，让细微调整更容易控制。

标注粗细无级调节

  • 几何、画笔、箭头、文本、马赛克不再依赖固定细/中/粗三档，按住 Command + 鼠标滚轮即可连续调节。
  • 调节后的粗细和字号会自动记忆，下次继续沿用当前手感。
  • 贴图标注画布同步支持同一套快捷调节逻辑，截图编辑和贴图编辑体验保持一致。

手感优化

  • 降低 Command + 鼠标滚轮灵敏度，避免轻轻滚一下就突然变得很粗。
  • 线条、箭头、画笔、马赛克每次滚动的最大变化被收窄，方便逐步调到合适粗细。
  • 文本字号滚轮调节同步降速，避免字号跳变过大。

兼容与同步

  • 连续粗细和字号写入本地偏好设置，支持小数精度保存。
  • 设置同步导出/导入增加连续粗细和字号字段，后续跨设备同步时保留标注手感。
  • 保留旧版字号等级兼容逻辑，已有设置升级后不会丢失。

发布包

  • 发布正式版 LumaTrace-1.11.dmg。
