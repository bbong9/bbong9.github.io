先看能做什么

主页不再放空内容，直接把 SwiftShot 最常用能力按使用场景展示清楚。

截图工作流框选、窗口识别、工具栏、复制/保存/拖拽导出。 标注与隐私保护箭头、矩形、文本、序号、灰白块状马赛克和模糊。 OCR 与翻译框选范围后直接识别/翻译，支持多国语言和可编辑文本框。 长截图与录屏自动/手动长截图、拼接线修复、区域录屏和系统录音引导。 设置与权限快捷键、保存格式、权限状态、授权、更新和同步入口。 历史与隐私最近记录、隐私模式、默认目录清理和敏感信息边界。

推荐阅读顺序

  1. 安装与授权先完成屏幕录制/辅助功能权限，避免截图空白或长截图不能滚动。
  2. 截图工作流学会框选、工具栏、快捷键和导出方式，这是最高频功能。
  3. 编辑标注掌握文本、箭头、序号、马赛克和 Command 拖动选择。
  4. OCR/翻译/同步配置 Provider 和 API Key 后，再开启跨设备同步备份。
