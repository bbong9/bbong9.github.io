同步 GitHub Releases 的正式版本更新内容，只记录用户能感知的重要变化，方便回看每个版本做了什么。

v1.7 - 2026-06-21

重点：修复更新页、截图工具栏序号卡死、无意义浮窗，并强化 OCR / 翻译排版与橡皮擦体验。

更新与设置

  • 修复更新界面排版错位问题，按钮和当前版本信息分层显示，避免拥挤重叠。
  • 恢复「手动检查更新」按钮，可在设置页立即检查新版本。
  • 更新页继续隐藏更新源 / appcast 等开发配置，只保留用户需要的自动检查、手动检查和更新记录。
  • 正式版显示保持干净，只显示两段正式版本号；测试版才显示构建号。

截图工具栏

  • 修复序号工具里「#」点击后卡死的问题，改为非阻塞小弹窗输入下一个序号。
  • 序号工具增加重置序号、自定义下一个序号能力，并统一 1、(1)、① 等图标大小比例。
  • 删除马赛克菜单里无意义的第三个按钮，仅保留框选马赛克和自由涂抹。
  • 删除截图工具栏坐标 / 尺寸小窗，减少干扰；OCR / 翻译框选时只提示当前动作。
  • 橡皮擦工具增加小 / 中 / 大三种圆形擦除范围，选中后显示圆形预览，拖动即可清除标注。

识别、翻译与文本排版

  • 优化截图工具栏识别 / 翻译结果文本排版，减少无意义硬换行和碎片化断句。
  • OCR 结果面板增加更自然的段落合并、滚动和行距处理，阅读更接近正常文章。
  • 列表、冒号、明显段落间距等场景会保留换行，避免把结构化内容强行拼成一行。

长截图与隐私标注

  • 继续修复长截图拼接线，增加边缘线清理和拼接缝修复。
  • 优化马赛克 / 柔焦隐私遮罩的导出和预览一致性。
  • 补充多项自动化测试，覆盖更新页、OCR 排版、导出、长截图拼接和版本显示。

发布包

  • 发布正式版 SwiftShot-1.7.dmg。
  • 同步 SwiftShot-latest.dmg，作为固定最新版下载入口。
  • 同步 Sparkle 自动更新源，支持客户端检查更新。

v1.6 - 2026-06-21

重点：解决钥匙串反复弹窗，补齐截图标注反馈、序号工具和演示录屏兼容性。

重点修复

  • 修复 OCR 识别与翻译反复弹出 macOS 钥匙串密码的问题。
  • OCR / 翻译 API Key 改为 SwiftShot 本机加密凭据库保存，避免本地 ad-hoc 构建后 Keychain ACL / cdhash 变化导致系统反复要求输入钥匙串密码。
  • 微信同步登录 token、同步密码也改为本机加密凭据库，减少打开设置页时触发系统钥匙串授权的风险。

标注与截图体验

  • 优化截图工具栏工具选中反馈，切换工具后状态更明确。
  • 优化序号工具：统一样式尺寸，修复不带圈序号选中后仍带圈的问题，增加自定义序号与重置序号能力。
  • 菜单栏文本工具改为统一的 T 图标显示，比例与其他工具保持一致。
  • 持续优化文本、箭头、马赛克等标注的拖动和编辑体验。

录屏 / 快捷键 / 设置

  • 修复状态栏菜单打开时快捷键截图响应异常的问题。
  • 优化截图界面与系统截图/录屏展示场景的兼容性，便于录制项目演示视频。
  • 更新版本历史跳转到 Wiki 更新记录页面。

安装包与更新

  • 发布正式版 SwiftShot-1.6.dmg。
  • 同步 SwiftShot-latest.dmg，作为固定最新版下载入口。
  • 同步 Sparkle 自动更新源，支持客户端检查更新。

v1.5 - 2026-06-21

重点：冻结截图画面、优化长截图拼接、补齐 OCR / 翻译夜间模式和自动更新发布链路。

截图与长截图

  • 修复进入截图界面后视频、动态画面仍继续播放的问题：触发截图后立即冻结当前屏幕，保证截取内容与用户看到的画面一致。
  • 继续优化长截图拼接算法，降低拼接线、重叠错位与滚动区域误差。
  • 优化长截图进度窗口与截图流程稳定性。

识别与翻译

  • 识别与翻译窗口新增夜间模式，深色界面下输入框、结果区、按钮与文字颜色全部适配，避免白底刺眼或文字不可见。
  • 优化 OCR/翻译结果面板的输入区域、复制按钮、排版与可读性。
  • 修复/改善翻译面板在深色外观下仍显示白色页面的问题。

设置与外观

  • 外观命名统一为“日间模式 / 夜间模式”，去掉“夜间浅色”这类歧义文案。
  • 设置页新增自动更新相关入口，便于后续从更新源获取新版本。
  • 增加设置同步相关结构，为微信登录后同步 API、接口与偏好设置做准备。

自动更新与发布

  • 增加 Sparkle 更新源生成脚本，并同步 appcast 与 zip 更新包到服务器。
  • GitHub Releases 同步上传正式版 DMG、latest DMG、appcast.xml 与自动更新 zip，保留历史版本并固定 latest 入口。
  • 优化正式版发布脚本，正式版继续使用两段版本号。

Wiki 与教程

  • 优化 GitHub Pages / Wiki 首页与功能教程排版。
  • 删除不必要的“发布与打包”教程页，聚焦安装、截图工作流、长截图、识别翻译、设置权限与隐私历史说明。
  • 增加截图工作流与设置页面配图，主页和教程页更直观。

授权与同步服务

  • LicenseCenter 增加设置同步备份相关接口与数据库迁移，为账号同步配置做准备。
  • 完善部署示例与接口说明。

v1.4 - 2026-06-20

重点：翻译 / OCR 面板细化到真实可用状态，首次权限与授权展示更稳。

翻译 / 文本识别面板

  • 左上角改为 SwiftShot 项目 App 图标，去掉临时渐变图标。
  • 左右文本区域升级为明显的圆角文本框，增加浅色底、边框和阴影，编辑状态更清晰。
  • 翻译失败时不再只显示“翻译失败”，会直接展示具体失败原因，方便定位 API Key、网络、额度或服务响应问题。
  • 保留左右双栏可编辑、复制、交换和反向翻译能力，继续使用 macOS 默认窗口控制按钮。

权限 / 授权体验

  • 首次打开仅在必要权限未完成时弹出权限检查，完成后后续不再重复打扰。
  • 优化永久授权状态卡片与授权展示体验。
  • 增加授权后台备注编辑能力，便于维护授权信息。

发布包

  • 同步生成 SwiftShot-1.4.dmg。
  • 同步上传固定最新入口 SwiftShot-latest.dmg。

v1.3 - 2026-06-20

重点：OCR / 翻译直接使用当前截图框选范围，权限引导回到独立窗口。

  • 文本识别 / 文本翻译：截图框选后直接对当前选区执行 OCR / 翻译，不再进入编辑器二次框选。
  • 权限引导：首次/缺权限时改为独立权限引导窗口，不再塞进设置页；删除重复的系统录音卡片。
  • 安装包：继续使用现代 DMG 安装界面，并增强 Finder 布局写入容错，避免系统偶发错误导致安装界面退回默认白底。
  • 发布资产：同步提供 SwiftShot-1.3.dmg 历史版本包和固定最新入口 SwiftShot-latest.dmg。

v1.2 - 2026-06-20

重点：隐私打码、标注导出、拖动手感、权限卡片和现代 DMG 安装体验集中优化。

隐私打码与标注

  • 马赛克改为更高级的灰白块状样式，替代原来的色条效果。
  • 强化隐私保护强度，优化马赛克细 / 中 / 粗画笔比例。
  • 修复标注、序号、文本、马赛克导出位置不一致和反转问题。
  • 修复文本二次编辑时旧文字残留、重叠的问题。
  • 优化文本、箭头、框选、序号、马赛克等标注拖动手感。
  • 截图编辑时按住 Command 可快速选择并拖动标注对象。

权限与首次启动

  • 首次启动不再显示旧控制面板，直接打开设置页。
  • 权限页新增可拖动 SwiftShot 卡片，可直接拖进系统设置列表。
  • 屏幕录制、辅助功能、录屏与系统录音都增加了拖拽入口，避免点加号后一个个找 App。
  • 录屏设置页在系统声音权限区域也增加拖拽卡片。

安装包体验

  • 全新现代化 DMG 安装界面。
  • 移除右下角“安装说明”和 DMG 内的 txt 文档。
  • 图标重新居中到安装卡片内。
  • 背景图改为 Retina @2x 渲染，改善 Finder 安装界面字体发虚。
  • DMG 只保留 SwiftShot.app 与 Applications 拖拽入口。
  • 修复 DMG 打开后偶发退回 Finder 白底默认界面的问题，打包时强制写入并校验 Finder 背景、图标尺寸和图标位置。

发布与版本管理

  • GitHub Releases 改为只发布正式版本：1.1、1.2、1.3。
  • 每个正式 Release 都上传版本号 DMG 和固定最新入口 SwiftShot-latest.dmg。
  • 本次 v1.2 已同步为 GitHub Latest。
