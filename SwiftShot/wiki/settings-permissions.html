设置与隐私

设置与权限

设置页用于管理快捷键、保存格式、历史隐私、授权、微信同步、自动更新和高级能力。

设置页分区

常规外观、当前状态、权限与隐私摘要。
快捷键全局快捷键、截图界面功能键、程序快捷键。
保存与格式图片格式、自动保存、录屏格式和目录。
高级OCR、翻译、长截图参数和诊断。

权限说明

  • 屏幕录制：截图、录屏、长截图读取画面必需。
  • 辅助功能：窗口识别、目标窗口聚焦、长截图自动滚动需要。
  • 系统录音：录屏需要录系统声音时开启。

保存与格式建议

  • PNG：适合界面截图、文字截图和需要清晰边缘的内容。
  • JPG：适合照片类内容，文件更小。
  • WebP：适合发布到网页或 Wiki，体积和清晰度更均衡。

高级设置说明

  • OCR / 翻译：配置识别和翻译 Provider、模型、接口地址等。
  • 长截图：根据页面滚动速度调整拼接和滚动参数。
  • 诊断：遇到权限、接口或导出问题时，用于定位原因。

微信同步怎么理解

  • 微信只做登录身份确认，配置不会写进微信。
  • API Key 上传前用同步密码 AES-GCM 加密。
  • 换设备登录同一微信并输入同步密码即可恢复接口、模型、快捷键和 API Key。
  • 忘记同步密码不会影响本机设置，但旧备份无法解密。
