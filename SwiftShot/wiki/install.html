安装与首次授权

这里面向普通用户，按“下载 → 授权 → 验证”的顺序完成首次启动。

下载安装

  1. 打开 版本下载页。
  2. 下载最新 SwiftShot.dmg 或 SwiftShot-latest.dmg。
  3. 打开 DMG，把 SwiftShot.app 拖入“应用程序”。
  4. 从“应用程序”启动 SwiftShot，确认菜单栏出现图标。

为什么需要权限

  • 屏幕录制：读取屏幕内容，截图、长截图和录屏都需要。
  • 辅助功能：识别目标窗口、拖动权限卡片、自动滚动长截图。
  • 系统录音：只有区域录屏需要录系统声音时才需要。

首次授权流程

  1. 首次打开后会弹出独立权限检查窗口。
  2. 先点“打开设置”，进入系统隐私与安全性页面。
  3. 如果列表里没有 SwiftShot，把权限窗口里的 SwiftShot 卡片拖入对应列表。
  4. 开启权限后回到 SwiftShot，后续满足权限时不会再反复弹窗。

安装后验证

  • 能用快捷键唤起截图界面。
  • 截图不是黑屏或空白。
  • 长截图能识别目标窗口并开始滚动。
  • 录屏能正常保存到默认录屏目录。
