标注工具

常用标注都保留在截图工具栏中，选中后有明确反馈。

  • 箭头、矩形、画笔用于说明重点。
  • 文本和序号用于教程步骤。
  • 序号支持重置和自定义下一个数字。

隐私保护

马赛克采用更高级的灰白块状风格，适合遮挡头像、手机号、订单号和 API Key。

  • 框选马赛克适合大块区域。
  • 自由涂抹适合局部信息。
  • 橡皮擦支持小 / 中 / 大圆形范围清除。

选中和拖动

按住 Command 进入快速选择拖动模式，可选择文本、箭头、马赛克、序号等标注对象。

  • 鼠标移动到边缘时有可拖动反馈。
  • 拖动时预览和最终保存位置一致。
  • 二次编辑文本不再残留旧内容。
