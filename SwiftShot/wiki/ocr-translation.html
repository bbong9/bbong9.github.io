识别流程

在截图界面框选后直接执行 OCR，不需要进入里面再框选一次。

  • 截图框选范围就是识别范围。
  • 识别结果尽量保留段落、列表和冒号结构。
  • 减少无意义硬换行，让文本更像正常文章。

翻译流程

翻译面板保持左右双栏：左侧原文可改，右侧目标语言实时更新。

  • 左侧文本栏支持编辑。
  • 右侧结果支持复制全部。
  • 语言列表覆盖主流国家语言。

语言与 Provider

API Key、Base URL、模型名和 Provider 类型在设置页配置。

  • 配置后再使用翻译。
  • 失败时显示具体错误原因。
  • 深色模式下文字、输入框、按钮全部适配。

FAQ

翻译失败怎么查

先按配置、网络、文本长度三个方向排查。

检查 Provider 配置
  • 确认 Base URL、API Key、模型名和 Provider 类型。
  • 确认网络能访问对应接口。
  • 先用短文本测试，排除原文过长或格式异常。
为什么要显示具体错误

只显示“翻译失败”无法定位问题，所以 SwiftShot 会尽量把接口响应、网络错误或配置错误直接展示出来。
