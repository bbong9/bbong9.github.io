核心功能教程

OCR 与翻译

框选范围就是识别范围：不再让用户在识别窗口里重复框选。

识别流程

  1. 截图界面框选需要识别的区域。
  2. 点击 OCR 或按截图界面功能键 O。
  3. SwiftShot 对当前框选范围进行文字识别。
  4. 识别结果可复制，也可继续进入翻译。

翻译流程

  1. 框选文本区域后点击翻译或按 F。
  2. 左侧文本框可编辑原文，右侧实时显示对应翻译。
  3. 选择目标语言后，主流国家语言可直接切换。
  4. 失败时先检查 Provider、Base URL、API Key 和网络。

语言与 Provider

  • 本地 Vision OCR 优先，不需要上传图片。
  • 翻译只发送识别后的文本，不上传整张截图。
  • OpenAI、DeepL、DeepLX 等接口信息存入 Keychain。
  • 如果开启微信同步，API Key 会先用同步密码加密后再备份。

翻译失败怎么查

  • 确认接口地址是完整的 https://...。
  • 确认模型名和 Provider 类型匹配。
  • 确认 API Key 没有多余空格。
  • 换网络或临时关闭代理规则后重试。
